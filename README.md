# Tamanga club
Website for the Triangle Anime Manga Club

## Functions
 - Cataloguing and display of books that can be checked out
 - Polls for what book people would like to read next as a group

## Technologies
- Python
  - Django
    - Knox
    - AWS-Storage
- Javascript
  - React
    - React-redux
    - React-bootstrap
  - Webpack & Babel
- Heroku
